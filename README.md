# Projeto relâmpago da Alpha EdTech

Criação de um portfolio para apresentar meus projetos e minhas habilidades técnicas

## Requisitos:
1. HTML Puro + Semântico.
2. Css Puro.
3. Css organizado, responsivo, acessível e animado.
4. Design bem-feito, bem pensado e limpo.
5. Deve ser criado um protótipo utilizando a ferramenta [Figma](https://www.figma.com/design/).
6. Projeto deve ser guardado em um repositório e servido pelo github pages.

## Período para desenvolvimento:
- 8 dias, incluindo entrega diária de atividades

## Processo realizado no figma:
1. Escolha do color-scheme;
2. Escolha das especificações;
3. Criação do protótipo;
    1. Criação dos componentes enquanto montava o protótipo;

## Processo realizado no código puro:
1. Criação da estrutura com emmet puro;
    - constante mudança ao longo do projeto para se adequar as ideias.
2. Preenchimento dos dados;
3. Estilização
4. Refatoração da aplicação da estilização seguindo padrão ITCSS e BEM

## Conteúdo
A página contém 7 seções principais:
1. header
2. (main) sobre mim
3. (main) minhas filosofias
4. (main) meu conhecimento
5. (main) meus projetos
6. (main) meus hobbies
3. footer

Contém modo escuro e claro, para pessoas que não conseguem lidar com o contraste o tema escolhido entorno da minha preferência de cor;
Contém links para páginas externas;
